# Story Management System

A laravel-vue app with built-in role-based authentication.
You can manage your stories in this app.

## Installation

#### Client side
```bash
  cd client-side
  npm install
```
#### Server side
```bash
  cd server-side
  composer install
  php artisan key:generate
  php artisan migrate --seed
  php artisan passport:install
```
